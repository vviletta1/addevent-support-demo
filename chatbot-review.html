🤖 Chatbot Quality Review Log

This log highlights examples where a chatbot handling developer support failed to provide helpful responses. Each entry includes the original user message, the bot's reply, the issue identified, and a suggested fix.

User Message Bot Response Issue Suggested Fix
“How can I send reminders with the API?” “Please check your email settings in your account.” Bot misunderstood – user was asking about automation, not settings. Update chatbot to recognize "reminders" as an API topic and suggest scheduling email reminders via the API.
“How do I let users RSVP to events?” “Our system supports multiple time zones.” Irrelevant response – doesn't address RSVP. Add conditional logic to suggest event registration forms or RSVP API features when “RSVP” is mentioned.
“Where do I put the script for the add-to-calendar button?” “You can style your button with CSS.” Bot skipped the key question – user needed installation help, not styling. Prioritize including the base script tag and setup guide when user asks about setup.
“Can I embed a calendar in my app?” “We help users add events to their personal calendars.” Partial match – didn’t address the app integration directly. Clarify that developers can use iFrame or widget embed and provide link to widget setup doc.
